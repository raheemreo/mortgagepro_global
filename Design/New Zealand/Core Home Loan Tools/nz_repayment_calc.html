⚠️ Interest Only — Key NZ Rules
· Max IO period: typically 5 years for NZ owner-occupiers (1–2 yrs for investors)
· After IO period ends, P&I repayments increase significantly
· No equity built during IO period — property value changes affect LVR
· RBNZ LVR rules still apply: owner-occ 20% min deposit; investors 35%
· IO common for property investors under ring-fencing rules (IRD 2019)
